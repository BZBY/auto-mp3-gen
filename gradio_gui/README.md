# 🎭 动漫角色对话提取系统 - Gradio 5.x版

## 🚀 快速开始 (Windows)

### 1. 一键启动
```bash
# 双击运行
start.bat
```

### 2. 手动启动
```bash
cd gradio_gui
python start.py
```

## 🌐 访问信息

- **默认端口**: 28000
- **访问地址**: http://127.0.0.1:28000
- **自动打开**: 启动后自动打开浏览器
- **SSR模式**: 启用服务端渲染，瞬间加载

## 📋 依赖要求

### 必需:
- **Python 3.10+** (Gradio 5.x要求)
- **FFmpeg** (音频处理)

### 可选:
- **UVR5** (人声分离，可显著提高效果)

### 自动安装:
所有Python依赖包都会在首次运行时自动安装，包括：
- gradio>=5.9.0
- 音频处理相关库
- 机器学习相关库
